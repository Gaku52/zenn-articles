# Zenn記事デプロイ

Zenn記事・本のデプロイを自動化します。カバー画像の圧縮も含みます。

## タスク

1. **変更ファイルの確認**
   - `git status` で変更されたファイルを確認
   - 変更内容を表示

2. **カバー画像の処理**（該当する場合）
   - books/*/配下のcover.png または cover.jpg を検出
   - 1MB以上の場合、JPEG形式で70%品質に圧縮
   - config.yamlのcoverパスを更新

3. **Git操作**
   - 変更ファイルをステージング（.DS_Storeは除外）
   - コミットメッセージを自動生成
   - プッシュ

4. **デプロイ結果の確認**
   - Zennのデプロイが成功したか確認するよう促す

## 実行条件

- zenn-articlesディレクトリで実行
- gitリポジトリが初期化されている
- リモートリポジトリが設定されている

## 注意事項

- カバー画像は自動的に1MB以下に圧縮されます
- .DS_Storeファイルは自動的に除外されます
- コミットメッセージは変更内容から自動生成されます
