# Zenn本執筆開始

このコマンドは新しいZenn本の執筆を開始する際に使用します。

## 実行内容

1. **重要**: `.zenn-deployment-notes.md`を読んで、デプロイ時の注意事項を確認する
2. 既存の本の一覧を表示
3. 新しい本を作成するか、既存の本を編集するかを確認
4. 執筆作業を開始

## 注意事項

- 複数チャプターを同時に執筆する場合、**必ず全てを同じコミットに含める**
- 執筆完了後、**即座にコミット・プッシュ**する
- Zennのデプロイログを確認して、該当ファイルが反映されているか確認する
