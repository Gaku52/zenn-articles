# 新しいチャプターを執筆

このコマンドは既存のZenn本に新しいチャプターを執筆する際に使用します。

## 実行内容

1. **最重要**: `.zenn-deployment-notes.md`の内容を確認して表示する
2. 既存の本とチャプター一覧を表示
3. 執筆するチャプターを確認
4. チャプターの内容を執筆
5. **執筆完了後、即座にコミット・プッシュ**
6. Zennデプロイログの確認を促す

## 注意事項

### ⚠️ Zennデプロイの重要ポイント

- **複数チャプターを執筆する場合**: 全て同じコミットに含める
- **古いチャプターを修正する場合**: 必ず再コミット・プッシュする
- **執筆完了後**: 3-5分待ってZennのデプロイログを確認
- **文字数が少ない場合**: `.zenn-deployment-notes.md`のトラブルシューティングを参照
