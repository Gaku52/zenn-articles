# Next.js App Router完全ガイドへようこそ

本書は、Next.js 14/15の**App Router**を完全にマスターするための実践的なガイドです。

## 📖 本書の対象読者

- Reactの基礎を理解している方
- Next.jsを学びたい、または移行を検討している方
- モダンなフルスタック開発を学びたい方
- Server Componentsの実践的な使い方を知りたい方

## 🎯 本書で学べること

### Part 1: App Router基礎（第1〜3章）
- ファイルベースルーティングの完全理解
- 動的ルート、パラレルルート、インターセプトルート
- レイアウトとテンプレートの効果的な使い方

### Part 2: コンポーネント戦略（第4〜6章）
- Server Componentsの特性と活用法
- Client Componentsとの使い分け
- パフォーマンス最適化パターン

### Part 3: データフェッチング（第7〜9章）
- fetchとキャッシング戦略
- リバリデーション（時間ベース・オンデマンド）
- Server Actionsによるフォーム処理

### Part 4: API開発（第10〜11章）
- Route Handlersによるバックエンド開発
- PrismaによるDB統合
- RESTful API設計

### Part 5: 実践（第12〜13章）
- 実際のアプリケーション開発
- Vercelへのデプロイと本番運用

## ✨ 本書の特徴

1. **TypeScript完全対応**
   全てのコード例がTypeScriptで書かれています

2. **実測データに基づく最適化**
   パフォーマンス改善の効果を数値で示します

3. **失敗事例から学ぶ**
   よくある間違いとその解決策を詳しく解説

4. **コピペ可能なコード**
   そのまま使える実践的なコード例

## 🚀 前提知識

本書を最大限に活用するため、以下の知識があることを推奨します：

- **React**: Hooks（useState, useEffect）の基本
- **TypeScript**: 基本的な型定義
- **HTML/CSS**: 基礎的なマークアップ
- **Git**: 基本的なコマンド

## 📦 開発環境

本書のコード例は以下の環境で動作確認しています：

```
Node.js: 18.x以上
Next.js: 14.x / 15.x
TypeScript: 5.x
React: 18.x
```

## 💡 本書の使い方

1. **順番に読む**: 基礎から段階的に学べる構成です
2. **コードを実際に試す**: 手を動かすことが最良の学習法です
3. **失敗事例を確認**: 同じ間違いを避けられます
4. **実プロジェクトで活用**: 学んだことをすぐに実践に活かせます
