# はじめに

> Go言語をゼロから学ぶための完全ガイド

## この本について

この本は、**他のプログラミング言語を触ったことがある方**が、Go言語の基礎を体系的に身につけるためのガイドです。

Goは「シンプルで、速くて、読みやすい」を徹底的に追求した言語です。機能を増やすのではなく、**必要最小限の機能で最大限の表現力を得る**という設計思想は、一度理解すれば驚くほど快適な開発体験をもたらします。

### この本を読み終えると、こんなことができるようになります

```
Goの基本構文を使って実用的なプログラムが書ける
ポインタ・構造体・インターフェースを正しく使い分けられる
Goらしいエラーハンドリングが自然にできる
go mod によるプロジェクト管理ができる
go test でテストを書いて実行できる
他の人が書いたGoコードを読んで理解できる
```

### 何を学べるか

- ✅ Goの設計哲学と「なぜGoなのか」
- ✅ 開発環境の構築（Go, VS Code, go mod）
- ✅ 基本構文（変数、定数、型、`:=`）
- ✅ 制御フロー（if, for, switch, defer）
- ✅ 関数（複数戻り値、クロージャ、可変長引数）
- ✅ データ構造（配列、スライス、マップ）
- ✅ ポインタとメモリモデル
- ✅ 構造体・メソッド・レシーバ
- ✅ インターフェースと構造的部分型
- ✅ エラーハンドリング（error, errors.Is/As, panic/recover）
- ✅ パッケージとモジュール管理
- ✅ テストの書き方

### 想定読者

- **他の言語（Python, JavaScript, Java など）は書けるが、Goは初めてという方**
- **Goの基礎を体系的に理解したい方** — 「なんとなく動く」ではなく「なぜそう書くのか」を理解したい
- **バックエンド開発やCLIツール作成に興味がある方**
- **シンプルで保守しやすいコードを書きたい方**

### 前提知識

**何らかのプログラミング言語の経験があること。**

以下の概念を理解していれば十分です：

- 変数、関数、条件分岐、ループの基本概念
- ターミナル（コマンドライン）の基本操作

## 本書の構成

本書は4部構成・全12章です。**Chapter 01 から順番に**読み進めてください。

### 第1部：Go入門（Chapter 01-03）

Goの全体像を把握し、開発環境を整え、最初のプログラムを書きます。

- **Chapter 01**: Go言語とは何か — 設計哲学と「なぜGoなのか」
- **Chapter 02**: 環境構築 — Go, VS Code, go mod のセットアップ
- **Chapter 03**: 基本構文と型 — 変数、定数、基本型、`:=`

### 第2部：基本概念（Chapter 04-06）

プログラムの基本構造を学びます。

- **Chapter 04**: 制御フロー — if, for, switch, defer
- **Chapter 05**: 関数 — 複数戻り値、クロージャ、可変長引数
- **Chapter 06**: データ構造 — 配列、スライス、マップ

### 第3部：Goの核心（Chapter 07-10）

Goらしいコードを書くための核となる概念を学びます。ここが本書の最重要パートです。

- **Chapter 07**: ポインタ — 値渡しと参照、メモリモデル
- **Chapter 08**: 構造体とメソッド — struct、メソッド、レシーバ
- **Chapter 09**: インターフェース — 暗黙的実装、構造的部分型
- **Chapter 10**: エラーハンドリング — error, errors.Is/As, panic/recover

### 第4部：実践への橋渡し（Chapter 11-12）

プロジェクト構成とテスト、そして次のステップ。

- **Chapter 11**: パッケージとモジュール — go mod、可視性、プロジェクト構成
- **Chapter 12**: 次のステップ — 並行処理、Web開発、実践ガイドへ

## 学習の進め方

### 推奨学習スタイル

1. **順番に読む**: 各章は前の章の内容を前提としています
2. **手を動かす**: サンプルコードは必ず自分で入力して `go run` してください
3. **`go doc` を活用する**: 気になった関数はすぐ `go doc fmt.Println` で調べましょう
4. **「やってみよう」に挑戦する**: 各章末の練習問題で理解を確認できます

### コードサンプルについて

- すべてのコードサンプルは**実際に動作するもの**です
- **Go 1.24** をベースに記述しています
- 重要なポイントには**コメント**で解説を入れています
- `$ go run` で始まるコードは、ターミナルでの実行例です
