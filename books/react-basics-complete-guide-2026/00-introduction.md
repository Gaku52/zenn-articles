# はじめに

> Reactをゼロから学ぶための完全ガイド

## この本について

この本は、**React初学者**のために書かれた体系的な基礎ガイドです。

JavaScript の基礎を理解している方が、Reactを使った現代的なフロントエンド開発を学べるように構成されています。

### 何を学べるか

この本を読むことで、以下のスキルが身につきます：

- ✅ Reactの基本概念と哲学
- ✅ 開発環境の構築方法
- ✅ JSXの書き方と理解
- ✅ コンポーネントベースの設計
- ✅ Props と State の使い分け
- ✅ Hooksを使った状態管理
- ✅ イベント処理とリスト表示
- ✅ カスタムフックの作成

### 想定読者

この本は以下のような方を対象としています：

- **HTML/CSS/JavaScriptの基礎を理解している**
- **Reactを初めて学ぶ、または基礎から学び直したい**
- **体系的にReactを理解したい**
- **実務でReactを使う予定がある**

### 前提知識

この本を読む前に、以下の知識が必要です：

**必須**:
- HTML/CSSの基礎
- JavaScriptの基礎（変数、関数、配列、オブジェクト）
- ES6の基本構文（アロー関数、分割代入、スプレッド構文）

**推奨**:
- Node.js/npmの基礎知識
- ターミナル/コマンドラインの基本操作

## 学習の進め方

### 推奨学習スタイル

1. **順番に読む**: 各章は前の章の内容を前提としています
2. **手を動かす**: サンプルコードは実際に書いて動かしてください
3. **繰り返す**: 理解が曖昧な箇所は何度も読み返しましょう
4. **実践する**: 小さなアプリを作って学んだことを応用しましょう

### コードサンプルについて

- すべてのコードサンプルは**実際に動作するもの**です
- **TypeScript**と**JavaScript**の両方の例を提示します
- 重要なポイントには**コメント**で解説を入れています
