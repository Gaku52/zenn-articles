# はじめに

> 基礎を固めた次のステップへ — Go実践開発ガイド

## この本について

この本は、**Go言語の基礎を習得した方**が、実践的な開発スキルを身につけるためのガイドです。

「Go言語 基礎から始める完全ガイド 2026」（姉妹本）で学んだ基本構文・ポインタ・構造体・インターフェース・エラーハンドリングの知識を土台に、**並行処理・Web開発・CLI開発・本番運用**まで一気にカバーします。

### この本を読み終えると、こんなことができるようになります

```
goroutineとchannelを使った並行処理プログラムが書ける
net/httpやGin/Echoを使ってWeb APIサーバーを構築できる
データベース接続とgRPCによるサービス間通信を実装できる
cobraを使った本格的なCLIツールを開発できる
ジェネリクスを使って型安全な汎用コードが書ける
pprofによるパフォーマンスチューニングができる
Dockerコンテナ化とデプロイパイプラインを構築できる
```

### 前提知識

**Goの基礎文法を理解していること。** 具体的には：

- 変数、関数、制御フロー（if, for, switch, defer）
- スライス、マップの操作
- ポインタと構造体
- インターフェースの仕組み
- エラーハンドリング（`error`, `errors.Is/As`）
- `go mod` によるプロジェクト管理

これらは姉妹本「Go言語 基礎から始める完全ガイド 2026」でカバーしています。

## 本書の構成

本書は3部構成・全14章です。興味のある部から読むこともできますが、**第1部は全ての基盤**となるため、最初に読むことを推奨します。

### 第1部：並行処理（Chapter 01-04）

Goの最大の強み — 並行処理を基礎から実践パターンまで。

- **Chapter 01**: goroutineとchannel — 軽量スレッドとメッセージパッシング
- **Chapter 02**: 同期プリミティブ — Mutex, WaitGroup, Once, sync.Map
- **Chapter 03**: 並行パターン — Worker Pool, Fan-out/Fan-in, Pipeline
- **Chapter 04**: Context — キャンセル、タイムアウト、リクエストスコープ

### 第2部：Web開発（Chapter 05-09）

Go最大のユースケース — Webサーバー開発の全体像。

- **Chapter 05**: net/http — 標準ライブラリだけで作るWebサーバー
- **Chapter 06**: Gin / Echo — 人気フレームワークの使い分け
- **Chapter 07**: データベース — database/sql, GORM, sqlx
- **Chapter 08**: gRPC — Protocol Buffersとサービス間通信
- **Chapter 09**: テスト — httptest, モック, テーブル駆動テスト

### 第3部：ツールと実践（Chapter 10-14）

開発ツールから本番運用まで。

- **Chapter 10**: CLI開発 — cobra, viper, クロスコンパイル
- **Chapter 11**: ジェネリクス — 型パラメータ、制約、実践パターン
- **Chapter 12**: プロファイリング — pprof, trace, ベンチマーク
- **Chapter 13**: デプロイ — Docker, CI/CD, クラウドデプロイ
- **Chapter 14**: ベストプラクティス — コード設計、プロジェクト構成、チーム開発

## 学習の進め方

### 推奨学習スタイル

1. **コードは必ず手で打つ**: コピペではなく、自分で入力して `go run` してください
2. **「やってみよう」に挑戦する**: 各章末の演習で理解を確認できます
3. **小さなプロジェクトを作る**: 学んだことを組み合わせて実際のツールを作ってみましょう

### コードサンプルについて

- すべてのコードサンプルは **Go 1.24** をベースに動作確認しています
- Go 1.22+ で導入された `net/http` の新ルーティング構文を使用しています
- ジェネリクスでは `cmp.Ordered`（標準ライブラリ）を使用しています
